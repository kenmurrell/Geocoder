How to use the Geocoder:

  1. Make sure you have Python 3.5 installed with pip. The instructions for their installation are in C:\Analytics\Software\Phyton\Python35
  2. Run the bat file run.bat
  3. When the program promps you, upload the csv file of addresses/postalcodes you wish to geocode in the format specified
  4. Wait through the loading screen; a time estimate is dispalyed
  5. Download the file with the "download!" button

Keep in mind only 2,500 records total can be geocoded per day
